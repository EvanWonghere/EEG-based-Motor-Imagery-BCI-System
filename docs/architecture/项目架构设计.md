## 一、项目概述与设计目标

### 1.1 项目简介

本项目为本科毕业设计，课题为**基于脑电运动想象的脑机接口系统研究**。系统采用「离线训练 + 数据回放仿真」架构，通过 Python 后端完成 EEG 信号的预处理、特征提取、模型训练与评估，再经 Lab Streaming Layer（LSL）协议将分类结果实时传输至 Unity 3D 前端进行可视化反馈。

### 1.2 设计目标

| 目标 | 说明 |
|---|---|
| **完整性** | 覆盖 BCI 系统的全链路：数据加载 → 预处理 → 特征提取 → 分类 → 在线仿真 → 可视化反馈 |
| **可对比性** | 支持多种特征提取方法（CSP、FBCSP、频段功率）与多种分类器（LDA、SVM、EEGNet）的系统对比 |
| **可复现性** | YAML 配置驱动实验，固定随机种子，环境快照，确保实验结果可复现 |
| **可扩展性** | 模块化设计，新增数据集/特征方法/分类器只需实现对应接口并注册 |
| **论文支撑** | 每个模块直接对应论文章节，代码输出即论文所需的图表和数据 |

### 1.3 技术栈

| 层级 | 技术 |
|---|---|
| 信号处理 | MNE-Python, SciPy, NumPy |
| 传统机器学习 | scikit-learn（CSP, LDA, SVM, RF） |
| 深度学习 | PyTorch（EEGNet, ShallowConvNet） |
| 实时通信 | pylsl（Lab Streaming Layer） |
| 可视化 | Matplotlib, MNE 脑地形图 |
| 配置管理 | PyYAML, python-dotenv |
| 前端 | Unity 6000+ / C# / LSL4Unity |
| 环境管理 | Conda + pip |

### 4.2 数据管理层 (`src/data/`)

#### 设计理念

将当前 `train_model.py` 中 500+ 行混杂的数据加载逻辑，拆分为统一接口下的独立加载器。每个数据集一个类，实现同一抽象基类。

#### 抽象基类

```python
# src/data/base.py
from abc import ABC, abstractmethod
import mne

class EEGDataset(ABC):
    """所有 EEG 数据集的统一抽象接口。"""

    def __init__(self, data_dir: Path, subjects: list[int] | None = None):
        self.data_dir = data_dir
        self.subjects = subjects or list(range(1, self.n_subjects + 1))

    @abstractmethod
    def load_raw(self, subject_id: int, session: str = "session_T") -> mne.io.Raw:
        """加载指定被试和会话的原始 EEG 数据。"""
        ...

    @abstractmethod
    def get_event_id(self) -> dict[str, int]:
        """返回事件 ID 映射，如 {'left_hand': 769, 'right_hand': 770}。"""
        ...

    @property
    @abstractmethod
    def n_subjects(self) -> int:
        """数据集中的被试总数。"""
        ...

    @property
    @abstractmethod
    def sfreq(self) -> float:
        """采样率 (Hz)。"""
        ...

    @property
    @abstractmethod
    def ch_names(self) -> list[str]:
        """EEG 导联名称列表。"""
        ...
```

#### FBCSP (`fbcsp.py`) — 本项目重点

```python
class FBCSPExtractor:
    """
    滤波器组共空间模式（Filter Bank Common Spatial Patterns）。

    这是 BCI Competition IV 冠军方案的核心算法，也是本项目的重点改进方法。

    原理：
    1. 将 EEG 信号通过多个带通滤波器，分解到不同子频带
       （如 4-8Hz, 8-12Hz, ..., 28-32Hz）
    2. 在每个子频带上分别应用 CSP，提取空间特征
    3. 拼接所有子带的 CSP 特征（如 7 子带 × 4 组分 = 28 维）
    4. 使用特征选择算法（互信息 / mRMR）选出最具判别力的特征子集

    相比普通 CSP 的优势：
    - 不需要预先确定最优频带，自动从多个子带中选择
    - 能同时利用不同频率范围的空间信息
    - 对被试间的最优频带差异具有更好的鲁棒性
    """

    def __init__(
        self,
        filter_bands: list[tuple[float, float]],
        n_components: int = 4,
        selection_method: str = "mutual_info",
        n_select: int = 8,
        sfreq: float = 250.0,
    ):
        self.filter_bands = filter_bands
        self.n_components = n_components
        self.selection_method = selection_method
        self.n_select = n_select
        self.sfreq = sfreq
        self.csp_list = []            # 每个子带一个 CSP 对象
        self.selector = None          # 特征选择器

    def fit(self, X: np.ndarray, y: np.ndarray) -> "FBCSPExtractor":
        """
        训练 FBCSP：
        1. 对每个子频带滤波 → CSP.fit()
        2. 拼接所有 CSP 特征
        3. 用互信息选择最优的 n_select 个特征
        """
        ...

    def transform(self, X: np.ndarray) -> np.ndarray:
        """
        变换新数据：
        1. 对每个子频带滤波 → CSP.transform()
        2. 拼接 → 特征选择 → 输出
        """
        ...

    def get_selected_band_info(self) -> list[dict]:
        """返回被选中的特征来自哪些子频带，用于分析和论文写作。"""
        ...
```

```python
# src/models/base.py
from abc import ABC, abstractmethod

class BaseModel(ABC):
    """所有模型的统一接口。"""

    @abstractmethod
    def fit(self, X: np.ndarray, y: np.ndarray) -> None: ...

    @abstractmethod
    def predict(self, X: np.ndarray) -> np.ndarray: ...

    @abstractmethod
    def predict_proba(self, X: np.ndarray) -> np.ndarray:
        """返回类别概率，用于 ROC 曲线和置信度判断。"""
        ...

    def save(self, path: Path) -> None: ...

    @classmethod
    def load(cls, path: Path) -> "BaseModel": ...

# 模型注册表
MODEL_REGISTRY = {}

def register_model(name: str):
    """装饰器，将模型类注册到全局注册表中。"""
    def wrapper(cls):
        MODEL_REGISTRY[name] = cls
        return cls
    return wrapper
```

#### EEGNet (`eegnet.py`)

```python
@register_model("eegnet")
class EEGNetModel(BaseModel):
    """
    EEGNet — 轻量级 EEG 深度学习模型。

    参考文献：Lawhern et al., "EEGNet: A Compact Convolutional Neural
    Network for EEG-based Brain-Computer Interfaces", 2018.

    网络结构：
    1. 时间卷积（Temporal Conv）：沿时间轴卷积，学习频率特征
    2. 深度卷积（Depthwise Conv）：沿空间轴卷积，学习空间滤波器（类似 CSP）
    3. 可分离卷积（Separable Conv）：组合时空特征
    4. 全局平均池化 → 全连接分类

    超参数：
    - F1: 第一层滤波器数（默认 8）
    - D: 深度卷积的深度乘子（默认 2）
    - F2: 可分离卷积滤波器数（默认 16, = F1 × D）
    - dropout: 丢弃率（默认 0.5）
    - kernel_length: 时间卷积核长度（默认 sfreq//2 = 125）

    注意：
    - 输入形状为 (batch, 1, n_channels, n_times)，不需要手动特征提取
    - 使用 AdamW 优化器 + 学习率调度 + 早停策略
    """
```

```
  │  +   │     │ ← 或 →  │   │(接收分类) │    │(正确/误)│   │      │

    固定十字     方向提示        Python 发送      显示结果     空白间隔

```

#### 核心脚本设计

**`LSLManager.cs`** — LSL 连接管理
```
职责：
- 搜索名为 "BCI_Control" 的 LSL 数据流
- 管理 StreamInlet 的生命周期（连接/断开/重连）
- 提供 C# 事件（event）供其他脚本订阅
- 显示连接状态 UI（已连接/等待连接）
```

**`MarkerReceiver.cs`** — 标记接收
```
职责：
- 在 Update() 中轮询 LSL Inlet
- 解析接收到的标记字符串（"Left", "Right" 等）
- 通过 UnityEvent 广播给 TrialController
```

**`TrialController.cs`** — 试次控制
```
职责：
- 管理试次的状态机（准备 → 提示 → 想象 → 反馈 → 休息）
- 控制每个阶段的持续时间
- 记录每个试次的真实标签和预测结果
```

**`FeedbackManager.cs`** — 反馈可视化
```
职责：
- 接收分类结果，驱动视觉反馈
- 进度条反馈：左/右方向的填充动画
- 可扩展为其他反馈形式（物体移动、颜色变化等）
```

**`SessionManager.cs`** — 会话管理
```
职责：
- 配置实验参数（试次数、各阶段时长）
- 记录整个会话的日志（CSV 格式）
- 统计在线准确率
```

```python
# 内部流程
def main(config_path: str):
    # 1. 加载并合并配置
    config = load_config(config_path)
    set_seed(config["seed"])

    # 2. 创建数据集
    dataset = create_dataset(config["dataset"]["name"], ...)

    # 3. 对每个被试训练
    results = {}
    for subject_id in config["dataset"]["subjects"]:
        # 3a. 加载原始数据
        raw = dataset.load_raw(subject_id)

        # 3b. 预处理
        pipeline = PreprocessingPipeline(config["preprocessing"])
        epochs = pipeline.run(raw, dataset.get_event_id())

        # 3c. 特征提取 + 训练 + 评估
        X, y = epochs.get_data(), epochs.events[:, -1]
        result = within_subject_cv(dataset, pipeline, extractor, model, ...)
        results[subject_id] = result

        # 3d. 保存模型（在全量数据上重新训练）
        model.fit(extractor.fit_transform(X, y), y)
        model.save(output_dir / f"sub{subject_id}.pkl")

    # 4. 保存结果和配置快照
    save_results(results, output_dir)
    save_config_snapshot(config, output_dir)
```
